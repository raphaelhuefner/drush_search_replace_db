### About:

drush_search_and_replace_db is an extension to drush which allows you to search and replace text in the whole DB.
	
### Usage
	
drush search-and-replace-db --search searchstring --replace replacestring [--search searchstring --replace replacestring [--search searchstring --replace replacestring [...]]]

Arguments:
 -s searchstring       At least one instance required. String to be searched.
 -r replacestring      At least one instance required. String acting as a replacement.
